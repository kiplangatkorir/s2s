## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.

## Deploying to Vercel

This frontend project is built with Next.js and pre-configured for Vercel deployment.

1. Push this code to a Git repository (GitHub, GitLab, or Bitbucket).
2. Create a new project on Vercel.
3. Import your repository.
4. The Framework Preset will be automatically detected as Next.js.
5. Make sure the root directory is set to `frontend/` if the repo contains both frontend and backend.
6. Set the appropriate environment variables if needed (e.g., if you have a hosted backend URL for WebSockets).
7. Deploy!
